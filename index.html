Events

Blinders

Blank Coding

Here comes our first event known as 'Blinders' where the participants code with a minimum number of errors. The twist is that they do it with their computer screens turned off! Something new and exciting ain't it?!

Bug out

Debugging event

the name of the event speaks for itself. You will be given a problem and told to bug it out! Now keep in mind that you, the participants will be penalized for changing the entire problem or writing your solutions. Keep your eyes open to what the buggy code has to show!

Web Paged

Site Development

This event's proceedings start as the competitors receive a professional brief and are instructed to build a user-friendly, contemporary website. LIVE! You are being watched!

Hubworks

GitHub project

In this event the participants are given the GitHub platform in which they'll have to search for a project to do and show the results to the respective person in charge (Jury)! Every search of yours will lead you closer to your success!

Diss-Cuss

Pannel Discussion

At 'Diss-Cuss' we have a panel discussion on setting a topic upfront and the participants get to speak on the spot. Kindly note that it is a knock-out round, so think before you speak. Every word is being heard!
